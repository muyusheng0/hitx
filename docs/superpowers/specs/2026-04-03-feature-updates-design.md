# 同学录网站功能改进设计方案

## 概述

对同学录网站的5项功能进行改进，包括图片交互、评论区、系统消息通知、页面体验和首页改版。

---

## 1. 留言板照片点击放大

### 方案
使用现有的灯箱（lightbox）组件，在留言板的图片上添加点击事件触发灯箱显示。

### 实现细节
- 留言板留言中的图片添加 `onclick="showLightbox(this.src)"` 事件
- 图片显示尺寸限制：`max-width: 200px; max-height: 200px`
- 点击图片后打开灯箱显示原图
- 点击遮罩或关闭按钮关闭灯箱

### 文件修改
- `templates/lyb.html` - 修改图片HTML添加点击事件

---

## 2. 评论区默认展开

### 方案
将评论区的默认 `display: none` 改为 `display: block`，移除"点击展开"的交互逻辑。

### 实现细节
- 留言发布后，评论区直接显示
- 用户可通过点击"收起评论"按钮关闭显示（可选功能）

### 文件修改
- `templates/lyb.html` - 修改评论区默认显示状态
- `static/js/main.js` - 移除 `toggleComments` 中默认展开逻辑

## 3. 系统消息通知功能（新增）

### 数据库设计
```python
# notifications 表
{
    'id': int,           # 通知ID
    'recipient': str,    # 接收人姓名
    'sender': str,       # 发送者姓名（可为空）
    'type': str,         # 'comment' | 'like_message' | 'like_photo' | 'like_video' | 'shout'
    'ref_id': int,       # 关联ID（留言ID/照片ID/视频ID等）
    'content': str,      # 通知内容摘要
    'is_read': bool,     # 是否已读
    'created_time': str   # 创建时间
}
```

### API接口

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/notifications` | GET | 获取当前用户通知列表 |
| `/api/notifications/count` | GET | 获取未读消息数量 |
| `/api/notifications/mark_read` | POST | 标记消息为已读 |

### 前端交互
- 登录用户右上角显示 🔔 图标 + 红点数字
- 点击图标弹出下拉列表，显示最新通知
- 每条消息显示：类型图标 + 内容摘要 + 时间
- 点击消息跳转到对应页面
- 点击"查看全部"跳转到消息页面（可选）

### 通知触发场景
| 场景 | 通知类型 | 跳转目标 |
|------|----------|----------|
| 评论自己的留言 | `comment` | 留言板该留言位置 |
| 点赞自己的留言 | `like_message` | 留言板 |
| 点赞自己的照片 | `like_photo` | 媒体相册 |
| 点赞自己的视频 | `like_video` | 视频页面 |
| 被其他同学喊话 | `shout` | 通讯录该同学卡片 |

### 文件修改
- `database.py` - 新增 notifications 表和相关函数
- `app.py` - 新增通知API接口，在点赞/评论/喊话时创建通知
- `templates/*.html` - 右上角添加通知图标和下拉列表

## 4. 页面底部提示 + 返回顶部按钮

### 方案
- 当用户滚动到页面底部时，底部显示"已加载全部内容"提示
- 返回顶部按钮：滚动超过 300px 后显示，点击返回顶部
- 按钮样式：固定右下角，圆形按钮，↑ 图标

### 适用页面
- 首页 `/`
- 通讯录 `/txl`
- 留言板 `/lyb`
- 媒体相册 `/media`

### 文件修改
- `static/css/style.css` - 添加返回顶部按钮样式
- `static/js/main.js` - 添加滚动检测和返回顶部函数
- `templates/*.html` - 添加按钮HTML和引用函数

## 5. 首页改版

### 新首页结构
```
┌─────────────────────────────────┐
│        顶部标题栏（固定）         │
├─────────────────────────────────┤
│                                 │
│      最新动态模块（大卡片）        │  ← 高调显目
│      - 轮播展示最新活动           │
│      - 每条活动可点击跳转          │
```

### 移除内容

- 更醒目的颜色/边框

- 支持触摸滑动

### 文件修改
- `templates/index.html` - 移除大字标题和统计模块，调整最新动态位置和样式
- `static/css/style.css` - 加强最新动态模块样式
