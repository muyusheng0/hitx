## 7. 留言板增加附加照片功能

**后端修改 (app.py)：**
```python
# 修改 read_lyb/write_lyb 支持图片字段
def read_lyb():
    messages = []
    if os.path.exists(LYB_FILE):
        with open(LYB_FILE, 'r', encoding='utf-8') as f:
            reader = csv.reader(f)
            for row in reader:
                if len(row) >= 4:
                    messages.append({
                        'id': row[0],
                        'nickname': row[1],
                        'content': row[2],
                        'time': row[3],
                        'image': row[4] if len(row) > 4 else ''  # 新增
                    })
    return messages

def write_lyb(messages):
    with open(LYB_FILE, 'w', encoding='utf-8', newline='') as f:
        writer = csv.writer(f)
        for m in messages:
            writer.writerow([m['id'], m['nickname'], m['content'], m['time'], m.get('image', '')])
```
