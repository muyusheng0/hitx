# 「时光八班」复古怀旧网站设计方案

## 设计愿景

在复古胶片美学的基调上，融入让人"WOW"的情感惊喜时刻。不靠炫技靠共鸣——让每个访问的同学都能感受到"这就是我的青春"。

**核心情感锚点**：
- 时光感 — 老照片、老物件、泛黄质感
- 仪式感 — 重要的时刻值得被"揭幕"
- 青春感 — 杏花、操场、毕业季

---

## 一、视觉基调

### 色彩系统

| 角色 | 色值 | 用途 |
|------|------|------|
| 主色 | `#2c3e50` | 深蓝灰（主背景底色） |
| 暖色强调 | `#e74c3c` | 暖红（重要交互、高光时刻） |
| 金色点缀 | `#f39c12` | 按钮、徽章、装饰线 |
| 背景主色 | `#1a1a2e` | 深色背景（替代原淡米色） |
| 背景副色 | `#16213e` | 卡片背景 |
| 泛黄效果 | `#d4a574` | 老照片边缘、日期印章 |
| 文字主色 | `#f5f5f5` | 白色文字 |
| 文字副色 | `rgba(255,255,255,0.6)` | 次要信息 |

### 字体

- 标题：`ZCOOL XiaoWei`（站酷小薇L）保持不变 — 这是情感锚点
- 正文：`Noto Sans SC` 保持不变
- 装饰元素：用 CSS 模拟老式打字机、印章效果

### 质感层

- 全局胶片颗粒噪点：`opacity: 0.03` 的 SVG 噪点叠加层
- 老照片边缘：虚化 + 暗角（CSS `box-shadow: inset`）
- 卡片边缘：微妙的锯齿/撕裂效果（CSS clip-path）

## 二、首页 — 时光入口

### 布局：Bento Grid

```
┌────────────────────────────────────────────┐
│  LOGO + 时光入口                    导航  │
├──────────┬──────────────────┬──────────────┤
│          │                  │              │
│  班级    │   时光通道       │   快捷      │
│  合影    │   (动态Banner)   │   入口      │
│  大卡片  │                  │   (3个小卡)  │
│          │                  │              │
├──────────┴──────────────────┴──────────────┤
│         新闻动态滚动区（宝丽来卡片）          │
├────────────────────────────────────────────┤
│              底部：同学录导航                │
└────────────────────────────────────────────┘
```

### 时光通道 Banner（杏花节动效）

- 深色渐变背景 + 微缩校园天际线剪影（SVG）
- CSS 动画：杏花瓣从屏幕上方缓缓飘落（`@keyframes` 控制 6-8 个花瓣的随机轨迹）
- 花瓣用 SVG 绘制，尺寸 15-25px，半透明
- 飘落时有微妙的左右摇摆（正弦曲线）
- 性能：纯 CSS，GPU 加速，无 JavaScript 负担

### 入场动效

- 页面加载时，所有卡片依次"飞入"（`animation: slideUp 0.6s ease-out`，stagger 间隔 100ms）
- 标题从左向右划过出现，有金属光泽扫过的效果

## 三、相册 — 时光机模式

### 宝丽来相框展示

每张照片做成**宝丽来相纸**样式：

```
┌──────────────────────┐
│  ┌────────────────┐  │  ← 白色相纸边缘 (padding: 12px)
│  │                │  │
│  │   照片内容      │  │  ← 照片主体
│  │                │  │
│  └────────────────┘  │
│  2024.06 · 毕业季    │  ← 底部泛黄日期印章
└──────────────────────┘
   ↑ 下方有撕裂边缘效果（clip-path）
```

### 悬停交互

- 悬停时：卡片微微倾斜（`rotate(-2deg)` → `rotate(2deg)` 随机）
- 照片"抽出来"：图片从相框里放大 110%，有阴影层次
- 底部日期印章有印章滚动的动画

### 点击灯箱

- 点击后，照片从当前位置**放大展开**到全屏（FLIP 动画）
- 背景有老式投影仪的光束效果
- 关闭时照片"收回"到原位

### 分类标签

- 标签做成**胶卷孔**样式：`▣ 毕业季`
- 选中时有"胶卷拉动"的切换动画

## 四、留言板 — 邮票信封

### 留言卡片

每条留言像**贴在信封上的邮票**：

```
┌─────────────────────────────────┐
│  ┌───────────────────────────┐  │
│  │  张三 · 2024.06.15        │  │
│  │  ─────────────────────    │  │
│  │  怀念那段日子...           │  │
│  └───────────────────────────┘  │
│  ▣ 日常  ▣ 回忆                │  ← 邮票锯齿边 + 分类孔
└─────────────────────────────────┘
```

### 邮票边缘效果

```css
.stamp-border {
  clip-path: polygon(
    /* 顶部锯齿 */
    0% 4px, 4px 0%, 8px 4px, 12px 0%, ... 100% 4px,
    /* 右边正常 */
    100% 4px, 100% calc(100% - 4px),
    /* 底部锯齿 */
    calc(100% - 4px) 100%, calc(100% - 8px) calc(100% - 4px), ...
    0% calc(100% - 4px)
  );
}
```

### 悬停动效

- 划过时，邮票"被揭起"：阴影加深 + translateY(-4px) + scale(1.02)
- 揭起时有纸张剥离的微声效（可选）

### 新留言入场

- 新留言从下方"弹出"，像信从信封里抽出
- 旧留言被向下"挤"，给新留言让位

## 五、背景音乐 — 黑胶唱机

### 播放器造型

```
┌─────────────────────────┐
│  ┌─────────────────┐   │
│  │   🎵 NOW         │   │ ← 频道指示灯（闪烁）
│  │  那些年（汪苏泷） │   │
│  └─────────────────┘   │
│                         │
│   ◉───────────  ═══    │ ← 唱针 + 唱片（旋转）
│      ♭  ♭  ♭          │ ← 音箱装饰孔
└─────────────────────────┘
```

### 播放状态

| 状态 | 视觉效果 |
|------|----------|
| 播放中 | 唱片旋转（CSS `rotate` 动画），唱针落下 |
| 暂停 | 唱片停止，唱针抬起 |
| 切歌 | 唱片"快进"旋转效果，然后停止到新曲 |

### 展开播放列表

- 点击展开后，像老式点唱机一样抽出一列唱片
- 每张唱片有封面缩略图、歌曲名、时长
- 当前播放的唱片有光晕效果

## 六、通行证 — 登录页

### 设计

- 深色背景 + 星光粒子动效
- 输入框是"老式拨号电话"的拨盘样式（数字围绕圆环）
- 或者更简洁：复古信封 + 手写字体
- 提交按钮："确认入场"（像入场券撕票）

## 七、新闻卡片

### 设计

- 卡片像**旧报纸剪报**：微微泛黄 + 撕裂边缘
- 日期用**红色印章**样式：`📅 2024.06.15`
- 来源 URL 隐藏，显示来源名称（如"南岭校区官网"）
- 悬停时，剪报"从墙上取下"的动效

---

## 八、动效规范

### 全局

| 动效 | 参数 | 场景 |
|------|------|------|
| 入场 stagger | 100ms 间隔 | 列表加载 |
| hover 微交互 | 200ms ease | 卡片悬停 |
| 页面切换 | 300ms ease-out | 路由切换 |
| 弹窗展开 | 400ms cubic-bezier | 灯箱打开 |

### 性能要求

- 所有动画使用 `transform` 和 `opacity`，触发 GPU 加速
- 花瓣动效不超过 8 个元素
- 避免动画时触发重排（`layout`）
- `prefers-reduced-motion` 用户可关闭动画

## 十、实施顺序

### 第一批（立即可见，冲击感强）

1. **花瓣飘落 Banner** — 首页入口，最容易让人"哇"
2. **宝丽来相册** — 照片是最核心的情感内容
3. **胶片颗粒全局叠加** — 一行 CSS，整体质感提升

### 第二批（细节打磨）

4. **邮票留言板** — 增强互动感
5. **黑胶播放器** — 音乐是氛围担当
6. **新闻报纸剪报样式**

### 第三批（锦上添花）

7. **Bento 布局网格**
8. **入场 stagger 动画**
9. **登录页重新设计**

---

## 十一、技术要点

- 动效纯 CSS 实现（性能优先）
- 花瓣用 SVG 或 CSS 绘制，不使用图片
- 播放器状态用 CSS class 切换，JS 只负责切换 class
- 所有组件样式独立文件，不污染全局
- 移动端：花瓣动效降级（减少到4个），胶片颗粒降低密度
