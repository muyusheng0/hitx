# 微信小程序 UI 美化设计

## 1. 概述

将微信小程序的 UI 风格与网站保持一致，营造统一的用户体验。

## 2. 设计规范

### 配色方案
| 变量 | 色值 | 用途 |
|------|------|------|
| --primary | #2c3e50 | 导航栏、标题 |
| --secondary | #e74c3c | 强调色、按钮 |
| --accent | #f39c12 | 点缀、渐变 |
| --bg | #faf8f5 | 页面背景 |
| --card-bg | #fff | 卡片背景 |
| --text | #333 | 正文文字 |
| --text-light | #666 | 次要文字 |

### 样式规范

**卡片**
- 背景：白色
- 圆角：12px
- 阴影：0 4px 20px rgba(0,0,0,0.08)
- 内边距：24rpx

**按钮**
- 渐变：linear-gradient(135deg, #e74c3c, #f39c12)
- 圆角：8px
- 内边距：24rpx 32rpx

**输入框**
- 边框：1px solid #e0e0e0
- 聚焦边框：1px solid #e74c3c
- 圆角：8px

**头像**
- 圆形
- 尺寸：80rpx (列表) / 120rpx (详情)

## 3. 页面改造

### 3.1 app.wxss (全局样式)
- 替换 page 背景为淡米色
- 定义全局卡片样式
- 定义按钮样式

### 3.2 app.json (窗口配置)
- navigationBarBackgroundColor: #2c3e50
- navigationBarTextStyle: white
- tabBar 保持现有配色

### 3.3 首页 (index)
- 标题区：大标题 + 副标题，卡片式
- 统计区：卡片展示，图标 + 数字 + 文字
- 最新留言：卡片列表，每条留言为独立卡片

### 3.4 通讯录 (txl)
- 搜索框：圆角输入框样式
- 列表项：白色卡片，左侧头像 + 右侧姓名城市
- 卡片间距：20rpx

### 3.5 同学详情 (txl-detail)
- 头像：大尺寸居中
- 信息区：卡片式展示各项信息
- 按钮：渐变背景

### 3.6 留言板 (lyb)
- 输入区：白色卡片包裹
- 留言列表：卡片式，每条留言独立卡片
- 时间显示：灰色小字

### 3.7 相册 (gallery)
- 图片网格：2列瀑布流
- 图片：圆角 8px
- 点击效果：阴影加深

### 3.8 视频 (video)
- 视频卡片：白色背景 + 封面图 + 标题
- 播放按钮：居中图标

### 3.9 个人中心 (profile)
- 头像 + 姓名：居中卡片顶部
- 信息列表：卡片式
- 退出按钮：渐变背景

### 3.10 绑定页 (bind)
- 输入框：统一样式
- 按钮：渐变背景

## 4. 实现文件

```
miniprogram/
├── app.wxss          # 全局样式
├── pages/
│   ├── index/        # 首页
│   ├── txl/          # 通讯录
│   ├── txl-detail/   # 同学详情
│   ├── lyb/          # 留言板
│   ├── gallery/      # 相册
│   ├── video/        # 视频
│   ├── profile/      # 个人中心
│   └── bind/         # 绑定页
```

## 5. 注意事项

- 保持小程序原生组件可用性
- 适配不同屏幕尺寸
- 颜色值使用 rpx 单位确保适配
