# 吉大通信八班 同学录网站规格文档

## 1. 项目概述

- **项目名称**: 吉大通信八班同学录
- **项目类型**: 动态Web应用（Flask + HTML5 + CSS3 + JavaScript）
- **核心功能**: 同学通讯录管理、留言板、相册视频展示、背景音乐
- **目标用户**: 吉大通信工程专业八班全体同学

## 3. 视觉与交互规格

### 3.1 设计风格
- **主题**: 复古胶片+现代简约融合风格，温馨怀旧
- **主色调**: #2c3e50 (深蓝灰) + #e74c3c (暖红) + #f39c12 (金黄) 作为点缀
- **背景**: 淡米色纹理背景，模拟老照片质感
- **字体**: "ZCOOL XiaoWei" (站酷小薇) 用于标题，正文使用 "Noto Sans SC"

### 3.2 页面布局
- **导航栏**: 固定顶部，毛玻璃效果，包含：首页、通讯录、留言板、相册、视频、关于
- **音乐播放器**: 右下角悬浮，控制面板可展开/收起
- **内容区**: 最大宽度1200px，居中显示

### 3.3 页面结构

#### 首页
- 大标题动画入场效果 "吉大通信八班"
- 班级合照/视频轮播
- 同学人数统计卡片
- 最新留言预览（3条）
- 毕业年份倒计时

#### 通讯录页面
- 网格卡片展示，每个卡片包含：姓名、籍贯、电话、现居地
- 点击卡片展开详细信息
- 搜索/筛选功能（按姓名、籍贯）
- 编辑入口：姓名+学号双重验证后允许修改个人信息
- 照片墙效果展示

#### 留言板页面
- 时间轴式留言展示
- 留言输入框（昵称 + 内容）
- 表情选择器
- 留言字数限制提示

#### 相册页面
- Masonry瀑布流布局
- 图片灯箱放大效果
- 上传功能（需验证姓名+学号）
- 分类筛选（按年份/活动）

#### 视频页面
- 视频卡片网格展示
- 支持优酷、腾讯、B站等第三方视频嵌入
- 点击播放模态框

#### 关于页面
- 班级介绍
- 网站说明
- 版权信息

### 3.4 动画效果
- 页面切换淡入淡出
- 卡片悬停上浮+阴影加深
- 音乐可视化波形
- 打字机效果显示欢迎语
- 雪花/星星飘落背景粒子效果（可选开关）

## 4. 功能规格

### 4.1 个人信息编辑
- **验证方式**: 输入 姓名 + 学号
- **验证通过**: 显示编辑表单，允许修改电话、现居地、备注、头像
- **安全措施**: 单次验证有效期30分钟（session）

### 4.2 留言板
- 输入昵称和留言内容
- 自动记录留言时间
- 支持表情符号插入
- 留言后立即显示

### 4.3 图片上传
- 支持 jpg/png/gif 格式
- 最大 5MB
- 自动生成缩略图
- 上传需姓名+学号验证

### 4.4 视频管理
- 添加第三方视频链接（优酷、腾讯、B站）
- 自动识别视频平台并嵌入
- 支持视频标题和描述

### 4.5 音乐播放器
- 内置5首怀旧/班级风格背景音乐
- 支持播放/暂停/上一首/下一首
- 音量控制
- 播放列表可展开选择
- 记忆上次播放状态

## 6. 技术栈

- **后端**: Flask (Python)
- **前端**: HTML5 + CSS3 + Vanilla JavaScript
- **数据存储**: CSV文件
- **文件上传**: 本地存储
- **音乐**: HTML5 Audio API

## 8. 安全性考虑

- 姓名+学号双重验证修改个人信息的入口
- Session-based 临时授权
- 文件上传类型和大小限制
- 输入内容XSS过滤
